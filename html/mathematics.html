1. Find the domain of the given function: \( f(x) = \sqrt{4 - x} + \frac{1}{\sqrt{x^2 - 1}} \).

  • A. \((-∞, -1] ∪ (-4, 1]\)
  • B. \((-∞, -4) ∪ (-1, 1]\)
  • C. \((-4, -1) ∪ (1, ∞)\)
  • D. \((-∞, -1) ∪ (1, 4]\)
View Answer

Answer: B. \((-∞, -4) ∪ (-1, 1]\)

View Solution

The domain of \( f(x) \) is determined by the conditions under which the function is defined. The function \( f(x) = \sqrt{4 - x} + \frac{1}{\sqrt{x^2 - 1}} \) is defined if and only if the following conditions are met: 1. \( 4 - x \geq 0 \), which implies \( x \leq 4 \). 2. \( x^2 - 1 > 0 \), which implies \( x > 1 \) or \( x < -1 \). Combining these conditions, we get the domain of \( f(x) \) as \((-∞, -4) ∪ (-1, 1]\).
