1. Find the domain of the given function: \( f(x) = \sqrt{4 - x} + \frac{1}{\sqrt{x^2 - 1}} \).

  • A. \((-∞, -1] ∪ (-4, 1]\)
  • B. \((-∞, -4) ∪ (-1, 1]\)
  • C. \((-4, -1) ∪ (1, ∞)\)
  • D. \((-∞, -1) ∪ (1, 4]\)
View Answer

Answer: D. \((-∞, -1) ∪ (1, 4]\)

View Solution

1. Square Root Condition:

For \( \sqrt{4 - x} \), the expression inside the square root must be non-negative, as square roots of negative numbers are not real.

\[ 4 - x \geq 0 \]

Solving for \( x \):

\[ x \leq 4 \]

So, \( x \) must be less than or equal to 4.

2. Rational Expression Condition:

For \( \frac{1}{x^2 - 1} \), the denominator must not be zero, as division by zero is undefined. Therefore:

\[ x^2 - 1 \neq 0 \]

Solving for \( x \):

\[ x^2 \neq 1 \quad \Rightarrow \quad x \neq \pm 1 \]

Additionally, for \( \frac{1}{\sqrt{a}} \) to be defined, the expression under the square root must be strictly positive. Thus:

\[ x^2 - 1 > 0 \]

Solving for \( x \):

\[ x^2 > 1 \quad \Rightarrow \quad x > 1 \quad \text{or} \quad x < -1 \]

3. Combining the Conditions:

From the square root condition, we have \( x \leq 4 \). From the rational expression condition, we know \( x \neq \pm 1 \) and \( x > 1 \) or \( x < -1 \).

We can now summarize the domain of the function using a number line:

  • For the square root, \( x \) must be less than or equal to 4.
  • For the rational expression, \( x \) must exclude \( \pm 1 \) and must satisfy \( x > 1 \) or \( x < -1 \).

The domain can be visualized as:

\[ \begin{aligned} (-\infty, -1) & \quad \text{(excluding -1)} \\ (1, 4] & \quad \text{(including 4 but excluding 1)} \end{aligned} \]

4. Final Domain:

The domain of the function, when combining all conditions, is:

\[ (-\infty, -1) \cup (1, 4] \]

2. Evaluate the limit: \( \lim_{x \to 2} \frac{x^3-8}{4x^2-16} \).

  • A. 0.25
  • B. 0.75
  • C. 1.5
  • D. DNE
View Answer

Answer: B. 0.75

View Solution

The limit of the function can be obtained by typing it to the calculator and use test numbers for x. The test number should be close to the value that x is approaching. For example, if we use x = 1.999 as it approaches 2 from the left, the function will give us 0.75. On the other hand, if we use x = 2.001 as it approaches 2 from the right, the function will give us 0.75. Therefore, the limit of the function is 0.75. If the two test numbers give us different values, then the limit does not exist.

  • C. 1
  • D. DNE

Answer: D. DNE

Using the test numbers x = 0.999 and x = 1.001, we can see that the function approaches positive and negative infinity, respectively. Therefore, the limit does not exist.
